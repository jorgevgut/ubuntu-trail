1. Emacs personalizado
___

- Copiar el archivo .emacs a la carpeta HOME que se puede referenciar con ~
- El tema tango tango debe estar en la siguiente ruta
  ~/.emacs.d/vendor/tango-tango
  (~ representa el path del HOME)
